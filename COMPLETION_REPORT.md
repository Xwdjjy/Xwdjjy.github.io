## 🛠️ 技术细节

### 系统架构

```
┌─────────────────────────────────────┐
│ 前端显示层 (src/index.html)          │
└─────────────┬───────────────────────┘
              │
┌─────────────▼───────────────────────┐
│ 脚本层 (js/index-manager.js)         │
│ • 数据获取                          │
│ • DOM 生成                          │
│ • 事件绑定                          │
│ • 搜索过滤                          │
└─────────────┬───────────────────────┘
              │
┌─────────────▼───────────────────────┐
│ 数据层 (index.json)                  │
│ • items[] 条目                       │
│ • recent[] 更新                      │
└─────────────────────────────────────┘
```

### 数据结构

```javascript
{
  "items": [
    {
      "id": number,               // 唯一ID
      "title": string,            // 标题
      "description": string,      // 描述
      "type": "posts|projects|docs", // 类型
      "url": string,              // 链接
      "date"?: string,            // 可选：日期
      "meta"?: string,            // 可选：额外信息
      "tags"?: string[]           // 可选：搜索标签
    }
  ],
  "recent": [
    {
      "title": string,            // 更新标题
      "icon": string,             // 单个字符头像
      "time": string,             // 时间描述
      "url": string               // 链接
    }
  ]
}
```

### 类型定义

| Type | Icon | 用途 |
|------|------|------|
| `posts` | 📝 | 博客文章、教程 |
| `projects` | 🚀 | 代码项目、作品 |
| `docs` | 📚 | API文档、指南 |
