## 使用流程

### 添加新条目

```
1. 编辑 src/index.json
2. 在 items 数组添加条目
3. git add src/index.json
4. git commit
5. git push
6. 刷新网页看效果
```

### 更新最近动态

```
1. 编辑 src/index.json
2. 在 recent 数组修改
3. git add src/index.json
4. git commit
5. git push
6. 刷新网页看效果
```

## 优势

### 相比硬编码 HTML

- ❌ HTML: 需要手写每个条目
- ✅ JSON: 结构化数据，易于管理

### 相比数据库

- ❌ 数据库: 需要服务器、维护成本高
